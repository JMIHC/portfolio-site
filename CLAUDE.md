## Project Overview

Portfolio and workshop site for Johnny Cornyn (johnnycornyn.com). The homepage is a "Build Your Website" workshop guide; secondary pages are Work (portfolio) and Contact. Deployed on Netlify as a static SPA.

## Commands

- `npm run dev` — start dev server (Vite, port 5173)
- `npm run build` — production build
- `npm run start` — serve production build locally
- `npx playwright test` — run Playwright tests

## Architecture

**React Router v7** in SPA mode (`ssr: false` in `react-router.config.ts`). Vite + Tailwind CSS v4 (via `@tailwindcss/vite` plugin).

- `app/routes.ts` — route definitions (flat, manual config using `index()` and `route()`)
- `app/routes/` — page components (`home.tsx`, `work.tsx`, `contact.tsx`)
- `app/components/` — shared components (`Nav`, `Footer`, `CodeBlock`)
- `app/root.tsx` — root layout with `<Nav />` and `<Footer />` wrapping `<Outlet />`
- `app/app.css` — global styles and Tailwind `@theme` tokens
- `public/contact.html` — hidden form for Netlify Forms detection (required for the AJAX contact form to work)

## Styling

Tailwind v4 with custom theme tokens defined in `app/app.css` under `@theme`. Key tokens: `--color-bg`, `--color-text`, `--color-text-secondary`, `--color-text-muted`, `--color-border`, `--color-surface`. Monochrome palette. Font is IBM Plex Mono (`--font-mono`).

Base styles are in `@layer base` in `app/app.css` — links, headings, `<pre>`/`<code>`, and `<hr>` are styled globally there.

## Netlify

- Build: `npm run build`, publish: `build/client`
- SPA fallback redirect (`/* → /index.html`) in `netlify.toml`
- Contact form uses Netlify Forms: hidden HTML form in `public/contact.html` for detection, AJAX submit in `contact.tsx`

## Route Types

Each route file imports its generated types from `./+types/<route>` (e.g., `./+types/home`). These are auto-generated by React Router into `.react-router/types/`. The `~/*` path alias maps to `./app/*`.
